# QR Code Generator

This is Python application with graphical user interface to generate QRcodes.

## Installation

This application use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```bash
pip install pillow
pip install qrcode

```
You can only run the requirements.bat
